# ChatGPT-Vue3 Demo

本專案為面試作品，展示使用 Vue 3 + TypeScript 的實作能力。  
主要功能為串接 OpenAI API，實現一個簡易版的 ChatGPT 對話介面。

## 🔧 技術棧

- ⚙️ **Vue 3**（Composition API）
- 🛡 **TypeScript**
- 🎨 **Tailwind CSS**
- 🔗 **OpenAI API 串接**

## 📌 目的

此專案僅用於面試展示，強調前端框架運用與 API 串接實作，介面設計風格簡潔。

## 📎 使用方式

1. 複製 `.env.example` 並命名為 `.env`
2. 填入你的 `VITE_OPENAI_API_KEY`
3. 執行開發環境：

```bash
npm install
npm run dev
